 # 🔐 Password Strength Checker (Python)

## 📌 Description

A simple yet effective **Password Strength Checker** built using Python. This project evaluates the strength of a password based on essential security criteria and provides instant feedback to help users create strong and secure passwords.

This tool is designed as part of a **Cybersecurity Internship project**, focusing on improving password security awareness and best practices.

---

## 🚀 Features

* ✔ Checks minimum password length (8+ characters)
* ✔ Detects presence of:

  * Uppercase letters
  * Lowercase letters
  * Digits
  * Special characters
* ✔ Provides real-time feedback
* ✔ Easy-to-use command-line interface

---

## 🛠️ Technologies Used

* Python
* Regex (re module)
